# PingPongGame
It is a once popular game in the US, but also was called the first electronic game in the world.

## 项目特点
* 游戏过程中的碰撞具有对应的音效和视觉效果
* 具有单人玩家和双人玩家模式
* 游戏过程中难度逐渐增加

## 版本更新
### version 1.0

1. 单人操纵鼠标进行游戏
2. 速度逐渐加快

### version 2.0
1. 增加双人玩家模式（键盘字母键A、D键控制 Player1，方向键左、右键控制 Player2）
2. 调整游戏元素大小，比如挡板和弹球大小
